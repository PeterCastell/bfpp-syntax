# Brainfuck++ Syntax Highlighting and Toolkit

VS Code syntax highlighting and Toolkit for the Brainfuck++ (`.bfpp`) language.

### Features
* Syntax highlighting in .bfpp files
* Syntax highlighting in markdown `bfpp` code blocks
* "Initialize Project" and "Run Current File" commands
* Debug adapter for running projects.

Commands and debug adapter are made for the compiler, a [separate repo](https://github.com/PeterCastell/BrainfuckPlusPlus).
